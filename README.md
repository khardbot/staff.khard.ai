## Contexto do produto

App iOS para mentoria agêntica de estudantes de medicina preparando para residência.

**Pilares do design:**
- Acompanhamento contínuo (dashboard de progresso em tempo real)
- Feedback agêntico (animações de resposta do mentor IA)
- Gamificação leve (streaks, metas, celebrações)
- Legibilidade em sessões de estudo longas (tipografia, contraste, espaçamento)
